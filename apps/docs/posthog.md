## Events

| Event | Description | File |
|-------|-------------|------|
| `cta_get_started_clicked` | User clicks "Get Started" in the hero CTA section | `src/sections/cta.tsx` |
| `navbar_get_started_clicked` | User clicks "Get Started" in the top navbar | `src/components/navbar/index.tsx` |
| `docs_copy_markdown_clicked` | User copies a docs page as Markdown | `src/components/mdx-components/page-actions.tsx` |
| `docs_view_option_clicked` | User opens docs page in an external tool (GitHub, ChatGPT, etc.) | `src/components/mdx-components/page-actions.tsx` |
| `search_filter_changed` | User changes the docs search filter tag | `src/components/search-dialog.tsx` |
| `faq_item_expanded` | User expands an FAQ accordion item | `src/sections/faq.tsx` |
| `blog_post_viewed` | User views a blog post (top of conversion funnel) | `src/app/blog/$slug.tsx` |
| `docs_page_viewed` | User views a docs page (top of conversion funnel) | `src/app/docs/$.tsx` |
| `blog_card_clicked` | User clicks a blog post card from the listing page | `src/app/blog/-components/blog-card.tsx` |
